# Credits

## Libraries

- **[Three.js](https://threejs.org/)** (r128, planned upgrade to r160+) — MIT license. WebGL rendering, scene graph, post-processing.
- **[MediaPipe Hands](https://developers.google.com/mediapipe/solutions/vision/hand_landmarker)** by Google — Apache 2.0. Real-time hand-landmark detection.
- **[MediaPipe camera_utils](https://www.npmjs.com/package/@mediapipe/camera_utils)** — webcam frame loop helper.
- **[MediaPipe drawing_utils](https://www.npmjs.com/package/@mediapipe/drawing_utils)** — landmark / connector overlay rendering.

- **[Space Mono](https://fonts.google.com/specimen/Space+Mono)** by Colophon Foundry — SIL Open Font License. Used for HUD numerics and labels.
- **[Rajdhani](https://fonts.google.com/specimen/Rajdhani)** by Indian Type Foundry — SIL Open Font License. Used for headings and body text.

Both are loaded via Google Fonts.

## Textures

The planet, moon, and Saturn-ring maps in `textures/` are widely circulated public-domain or attribution-permitted maps derived from NASA imagery. Common sources:

- **[NASA Visible Earth](https://visibleearth.nasa.gov/)** — Earth, Moon, and Sun maps.
- **[NASA JPL Solar System Maps](https://nasa3d.arc.nasa.gov/models)** — Mars, Jupiter, Saturn, Uranus, Neptune.
- **[Solar System Scope textures](https://www.solarsystemscope.com/textures/)** — community-curated mirror of NASA-derived equirectangular maps, distributed under CC-BY 4.0.

If you redistribute this project, please verify each texture file against its upstream license and add specific attributions here.

## Inspiration

The HUD aesthetic borrows from sci-fi cockpit / mission-control conventions popularized by _Alien_, _Mass Effect_, and _Destiny_ UI design, filtered through a cyberpunk color palette (cyan + gold + magenta on near-black).
